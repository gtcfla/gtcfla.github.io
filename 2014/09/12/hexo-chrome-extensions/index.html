常用Chrome扩展（不定期更新）

2014-09-12   |   工作   |     |   热度 °C

自工作以来，Chrome是我认为最好用的浏览器，没有之一！拥有异常丰富的扩展插件应用商店，下面记录和推荐自认为值得珍藏备用的插件，以供参考和记录。

  1. LastPass：用于管理大量网站密码，给不同网站设置不同密码，支持自动登录和手机两步验证！只需记住一个密码即可，在线安全地存储您的密码。
  2. Adblock Plus：广告屏蔽扩展，可以屏蔽不少网站的广告，包括屏蔽Youtube视频广告、Facebook广告、横幅广告以及其他广告，还可以自定义规则，在网络广告泛滥的年代里，可以做到眼不见心不烦。该扩展是一个社区驱动的开源项目 ，有不少志愿者为其作出了贡献。
  3. Proxy SwitchySharp：天朝必备，用于安全上网的扩展，你懂的。
  4. IE Tab：快速切换为IE内核，可用于不支持Chrome网站，如网银，也可用作网站兼容IE调试。
  5. Firebug Lite for Google Chrome™：调试JS插件，Chrome上的Firebug，类似于Firefox浏览器的Firebug。
  6. 印象笔记·悦读：用来保存网上看到的自己关注的文章和同步印象笔记本利器，灰常好用

功能增强扩展：

  • 一键管理所有扩展：可以快速开启、禁用全部扩展，或者启用停止单个扩展；这样，平时可以禁用一不太常用的插件达到节省系统资源的目的，而需要用到时临时启动也很方便。
  • Context：和上边的功能类似，它不仅可以禁用/启用全部扩展，还可以让你对扩展进行分组，并且可以快速启用/禁用某个分组的扩展。
  • Search by Image (by Google)：通过图片搜索图片，可以在网页上任何的图片身上右键点击进行 Google 的“以图搜图”，缩略图找原图或找相似图片的利器！
  • Pushbullet – 子弹推送：可以快速在 iOS、Android 手机和电脑的 Chrome 之间互相传递文字、链接、图片等信息。
  • OneTab：将当前打开的所有标签页保存到一个页面（列表）以备稍后或下次再访问；同时可以大大减少 Chrome 打开大量标签页时的内存消耗！
  • XMarks – 跨浏览器收藏夹同步：虽然现在的浏览器都能同步收藏夹，但 XMarks 可以在 IE、FireFox、Chrome 等不同浏览器之间同步你的收藏夹，这样无论使用那款浏览器都能更加方便了。
  • Proxy SwitchySharp – 浏览器代理神器：可以轻松快捷地管理多个代理，并能快速地在代理之间切换。可以设置规则或使用 PAC 自动智能对指定的网站使用代理，一句话 Fan 士啬 必备。
  • Checker Plus for Gmail – 新邮件提醒：对于经常使用 Gmail 的用户来说非常实用，它能显示 GMail 和 Google 企业邮箱收件箱中的新邮件和联系人照片；通过桌面通知获得新邮件提醒；不打开Gmail网页就能收取、阅读、回复和删除邮件。
  • 网页截图 注释&批注：捕获整个网页页面或任何部分，矩形，圆形，箭头，线条和文字，模糊敏感信息，一键上传分享注释等
  • Fatkun图片批量下载：可以帮你找出当前网页中的所有图片，提供按分辨率、链接等筛选图片，是一个简单好用的批量下载图片扩展。
  • Tampermonkey- – 安装管理油猴脚本 (GreaseMonkey) ，异次元之前也曾介绍过油猴脚本。
  • Page Monitor – 监视网页变化并提醒：驻留后台并监控网页变化，一旦网页内容有所改变即会提醒你。
  • Incognito-Filter：让指定的网站在“隐身窗口”中打开，用来对付一些不太想留痕迹的网站
  • Downloadr- / Download Manager：在 Chrome 的工具栏右上方管理你的下载文件
  • Vimium- - 键盘流必备神器：异次元曾经介绍过的一款效率插件，可以用 VIM 的操作方式来操作 Chrome，Geek 们和键盘操作控必备。
  • APK downloader：无需帐号即可直接在 Google Play 商店中直接下载 Android 的 .apk 安装包，我们之前在这里有介绍过。
  • The QR Code Extension – 二维码：快速将当前的页面网址生成一个 QR 二维码供手机扫描。
  • Context Menu Search：Chrome 右键默认只能进行 Google 搜索，这枚插件可以为你的右键增加多个搜索引擎，百度、淘宝、必应、360等等，想用什么搜就用什么搜。

软件/服务增强扩展：

  • 印象笔记·剪藏：提到印象笔记大家肯定不会陌生，使用印象笔记剪藏扩展程序可以快速一键保存网页内容到印象笔记帐户中去。
  • 为知笔记网页剪辑器：为知笔记是国产笔记软件中的佼佼者，也是印象笔记强劲有力的对手。
  • OneNote Clipper：来自微软的 OneNote，也有着自己不少的特点。
  • WunderList – 奇妙清单：一款非常实用的跨平台 TODO 清单记事类 GTD 软件，可以很好地帮助你有条不絮地安排你的生活和工作。
  • Any.Do：和 WunderList 类似的另外一款跨平台的 TODO 清单软件。
  • QuickDrop：在 Chrome 上快速浏览 Dropbox 里的文件，并且可以让你将图片、文件直接下载到你的 Dropbox 账户中去

开发辅助类扩展：

  • User-Agent Switcher：可以随时切换浏览器的 User-Agent (UA/浏览器标识)，一键模拟各种浏览器和手机浏览器，开发者必备助手。
  • JSON-handle：对 JSON 格式的数据进行重新排版，以树形图样式更直观地展现 JSON 文档，并且支持实时编辑。
  • Postman：一个相当强大好用的 REST API & HTTP 请求调试工具，它能够发送任何类型的HTTP requests (GET, HEAD, POST, PUT..)，附带任何数量的参数+ headers；支持不同的认证机制（basic, digest, OAuth），接收到的响应语法高亮（HTML，JSON或XML）；Postman 可以简化测试和调试HTTP请求，是每个 Web 开发人员都值得安装的扩展。
  • Cache Killer：很多时候页面缓存会让是 Web 开发人员遇到麻烦，这个插件可以让你在加载网页前自动清除缓存。
  • Window Resizer：可以快速调整浏览器窗口的尺寸，用于观察网站在不同宽度下的显示效果，比较适合用于测试现在流行的响应式网页。
  • Eye Dropper：获取网页上某一点的颜色值
  • Regex Search：用正则表达式搜索网页内容，有时需要搜索特定条件的文本时非常有效。这个实际上不算开发工具，但一般只有开发人员才懂正则，所以放到这边来。
  • RegExp Tester App：正则表达式测试、调试以及替换的 Chrome 应用。
  • EditThisCookie：Chrome 上的网页 cookie 管理插件，您可以通过它添加，删除，编辑，搜索，锁定和屏蔽 cookies。
  • Referer Control：修改 HTTP Referer 来源地址
